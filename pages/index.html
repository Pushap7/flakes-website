Build better business software a lot quicker with flakes

Flakes is an Admin Template Framework
A combination of CSS Libraries, JavaScript Libraries and Design files that help you build business tools very quickly.

Flakes was designed to be highly functional software that is user friendly, mobile ready and intuitive to use. Flakes takes a no-nonsense approach to Admin Templates.

Design & Research

In an attempt to build the finest — most functional Admin Template Framework, I started looking at existing implementations. I studied the admin websites that companies built for their internal use and backend applications on the market.

Unsurprisingly, these admin websites were universally inferior to their public facing counterparts. The user experience is unpleasant to say the least. Mobile readiness nonexistent. And an aesthetic that is as old as the internet itself.

Flakes is different. Well designed from the ground up. An uncluttered interface that lets you get work done without jarring your sensibilities. Responsive so you can manage your business on the go. Built around a component architecture for easy development. And much more.

The Flakes aesthetic is sharp, uncluttered & task driven. Flakes is not influenced by trends because our customers intend to keep using it for many years.

Thousands of hours are spent on admin applications. That makes speed a feature and a priority. Flakes was designed to be weightless. It goes away with custom fonts, heavy images or slow loading libraries. All you are left with is a blazingly fast framework.
